## Quick Start

### Prerequisites

- Python 3.10+
- Node.js 18+ (optional, for development)
- API keys (optional for demo mode)

### Installation

```bash
# Clone the repository
git clone https://github.com/alpha-1-design/aether-pwa.git
cd aether-pwa

# Create virtual environment
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate     # Windows

# Install dependencies
pip install -r requirements.txt

# Run the server
python server.py
```

### Environment Variables

```bash
# Create .env file
cat > .env << 'EOF'
# API Keys (optional - demo mode works without)
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GOOGLE_API_KEY=...
OPENROUTER_API_KEY=...
TAVILY_API_KEY=...

# Server Configuration
PORT=5000
FLASK_DEBUG=false
FLASK_ENV=production

# Rate Limiting
RATELIMIT_DEFAULT=100 per day
RATELIMIT_STORAGE_URL=memory
EOF
```

### Access

- Local: `http://localhost:5000`
- PWA: Install from the app (offline capable)

## Deployment

### Vercel (Frontend)

1. Connect GitHub repo to Vercel
2. Set build command: `echo "Static site"` (no build needed)
3. Set output directory: `/`
4. Add environment variables
5. Deploy!

### Render (Backend)

1. Create Web Service
2. Connect GitHub repo
3. Set build command: `pip install -r requirements.txt`
4. Set start command: `python server.py`
5. Add environment variables
6. Deploy!

### Railway

```bash
railway init
railway add
railway up
```

### Docker

```bash
# Build
docker build -t aether-pwa .

# Run
docker run -p 5000:5000 \
  -e ANTHROPIC_API_KEY=your_key \
  -e OPENAI_API_KEY=your_key \
  aether-pwa
```

## API Reference

### Chat Endpoint

```http
POST /api/chat
Content-Type: application/json

{
  "messages": [
    {"role": "system", "content": "You are helpful."},
    {"role": "user", "content": "Hello!"}
  ],
  "model": "claude-3.5-sonnet"
}
```

### Fetch Endpoint

```http
POST /api/fetch
Content-Type: application/json

{
  "url": "https://api.example.com/data"
}
```

### MCP Endpoints

```http
GET  /api/mcp/list
POST /api/mcp/execute
GET  /api/mcp/status
GET  /api/mcp/history
```

### Response Format

```json
{
  "response": "AI response text...",
  "model": "claude-3.5-sonnet",
  "tokens_used": 1500,
  "cost": 0.002,
  "latency_ms": 1234
}
```
